Easythreed E3D Nano

I took the printer settings from the configuration file of the official E3D NANO program. In fact, the built-in profiles do not differ much from each other, only in layer thickness and speeds. Manual settings in the program do not work very well.
I needed these parameters in order to configure another slicer PrusaSlicer.

• Layer height: Custom - 0.2 | Fast - 0.2 | Standard - 0.2 | Optimal - 0.1
• Shell thickness: Custom - 1.2 | Fast - 0.8 | Standard - 1.2 | Optimal - 1.2
• Initial layer thickness: Custom - 0.3 | Fast - 0.3 | Standard - 0.3 | Optimal - 0.3
