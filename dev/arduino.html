Easythreed E3D Nano PrusaSlicer

These are my PrusaSlicer settings for Easythreed E3D Nano 3D printer. I print purely technical things (cases and mechanisms) on this printer, perhaps for art printing, these settings will not work.

Easythreed E3D Nano Settings

I took the printer settings from the configuration file of the official E3D NANO program. In fact, the built-in profiles do not differ much from each other, only in layer thickness and speeds. Manual settings in the program do not work very well. I needed these parameters in order to configure another slicer PrusaSlicer.

Automatic Laser Cat Toy (Arduino)

This is an automatic laser cat toy. It is enough to plug the USB cable into the power bank and place the toy on any hill, for example, a closet.
